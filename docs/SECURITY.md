# Security
## Modelo
Todo tráfico del navegador debe pasar por un proxy activo validado.
## Limitaciones
No hay anonimato absoluto ni protección total frente a fingerprinting avanzado.
## Riesgos
Proxies inseguros pueden inspeccionar tráfico y credenciales.
## Datos locales
Se guarda estado de proxies, proxy activo y preferencias locales.
## Credenciales
Se evita persistir contraseñas en claro (enmascaradas al guardar).
## Reporte
Reportar vulnerabilidades mediante issue privado o contacto de mantenedor.
